# 🎮 Simon Says Game

A browser-based memory game inspired by the classic "Simon Says," built with vanilla HTML, CSS, and JavaScript. The game generates a growing sequence of colors that the player must repeat correctly to advance levels.

## 🕹️ How to Play

1. Press any key on your keyboard to start the game.
2. Watch the sequence of colors that flash on screen.
3. Click the colored buttons in the same order to repeat the sequence.
4. Each correct round adds a new color to the sequence, increasing the difficulty.
5. If you click the wrong color, the game ends, your score is displayed, and the screen flashes red.
6. Press any key again to restart and try to beat your previous score.

## 🛠️ Built With

- **HTML** – game structure and button layout
- **CSS** – styling and flash animations
- **JavaScript** – game logic, sequence generation, and user input handling
